This repository contains my progress through the **Higher-Level Programming** track at **ALX Software Engineering**. Unlike the previous track which focused on low-level memory management with C, this curriculum focuses on high-level abstraction, object-oriented programming (OOP), databases, and web development using **Python**, **SQL**, and **JavaScript**.

All Python code follows the **pycodestyle** (formerly PEP 8) standard.
